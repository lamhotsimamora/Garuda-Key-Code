# Garuda-Key-Code


## DEMO
https://lamhotsimamora.github.io/Garuda-Key-Code/


## Syntax
```
_doWhen({
  keycode:_keyCode.enter,
  callback:function(){
  	setup('ENTER',_keyCode.enter);
  }
});
```
